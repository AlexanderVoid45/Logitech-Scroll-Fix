# Logitech Master 3S Scroll Fix

Отключает smooth/smart scrolling через реестр без Logi Options.

## Использование

Отключить умный скролл и установить 1 строку:
```powershell
.\fix_logitech_scroll.ps1
```

Отключить умный скролл и установить 0.1 строки (пиксели):
```powershell
.\scroll_0.1_line.ps1
```

Восстановление стандартных настроек (3 строки):
```powershell
.\restore_to_default.ps1
```

Требуются права администратора.
